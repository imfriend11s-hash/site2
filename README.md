# Операция

Текстовый квест в жанре научной фантастики / хоррор.

## Сюжет

Ты просыпаешься в заброшенной лаборатории. Компания по исследованию мутаций заманила людей под видом детского лагеря. Девочка Лена предупреждает: если врачи проведут операцию — вы превратитесь в зомби. Твоя задача — найти ключ, код и спасти всех.

## Как играть

1. Скачай архив `Operacia.zip`
2. Распакуй его
3. Запусти файл `Operacia.py` (двойным щелчком или через командную строку)

## Управление

- Вводи цифры (1, 2, 3) для выбора действий
- Отвечай «да» или «нет»
- Игра запоминает предметы в инвентаре
